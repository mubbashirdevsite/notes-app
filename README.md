# 📝 Modern Notes App

A powerful and modern **Notes Application** built using HTML, CSS, and Vanilla JavaScript with full CRUD functionality and Local Storage support.

---

## 🚀 Features

- Create notes
- Edit notes
- Delete notes
- Pin important notes
- Color labels (Yellow, Blue, Green, Pink)
- Live search
- Sorting (Newest, Oldest, A-Z)
- Copy notes (Clipboard API)
- Clear all notes with confirmation
- Created & updated timestamps
- Persistent storage using LocalStorage
- Empty state UI
- Smooth animations
- Responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
